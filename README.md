# 30 Days of React – The Deliberate Practice Journey

Welcome to my **30‑Day React Mastery Challenge**. This repository is a structured, hands‑on log of my daily progress in learning React from core foundations to advanced patterns. Every day follows a disciplined **3.5‑hour routine** that combines theory, code‑along, independent building, debugging, and documentation.

---

## 🧠 The Philosophy

> "Don't just watch tutorials. Write code. Break things. Fix them. Document the journey."

This is not a passive course. It's a deliberate practice system where each day builds on the previous one, ensuring deep understanding and long‑term retention.

---

## ⏱️ The 3.5‑Hour Daily Routine

Every single day is divided into six focused segments:

| Segment | Duration | Activity |
|---------|----------|----------|
| **Morning Review** | 15 min | Recap previous day's concepts and notes |
| **New Concept Study** | 45 min | Deep dive into theoretical foundations and official docs |
| **Code Along** | 30‑60 min | Follow a tutorial or guide to implement the new concept |
| **Mini Build** | 45 min | Independently build a small feature or component |
| **Debug & Refactor** | 30 min | Clean up code, fix errors, and improve structure |
| **GitHub Push & Notes** | 15 min | Document progress and maintain a daily commit streak |

---

## 🗓️ 30‑Day Curriculum Roadmap

| Week | Focus Area | Key Topics |
|------|------------|------------|
| **Week 1** | Core Foundations | JSX, Components, Props, State, Events, Conditional Rendering |
| **Week 2** | Advanced UI Patterns | Forms, Lifting State, useEffect, API Calls, Custom Hooks |
| **Week 3** | Routing & State Management | React Router, Context API, useReducer, Zustand |
| **Week 4** | Performance & Deployment | Memoization, Code Splitting, Testing, Deployment to Vercel/Netlify |
| **Final Days** | Capstone Project | Build and deploy a complete full‑stack‑like application |

## 🚀 Quick Start

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/nahid-77/React-30-days-Learning-Challenge.git

# Navigate into the folder
cd react-foundations

# Install dependencies
npm install

# Start the development server
npm run dev
```
